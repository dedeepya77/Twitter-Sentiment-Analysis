# Twitter-Sentiment-Analysis
Built a Twitter Sentiment Analysis system using Python, Scikit-learn, and NLTK to classify tweets as Positive, Negative, or Neutral. Extended analysis using Navarasa theory to detect nine emotions. Implemented and compared Multinomial NB, Gaussian NB, and KNN models using accuracy, classification reports, and confusion matrices.
